Minha experencia com Interfaces com HTML/CSS

Minha experiencia nessa materia foi tranquila, pensei que seria um bixo de 7 cabeças criar interfaces mas é mais facil do que parecia, com minhas pesquisas aprendi a usar flexbox , o que deixou tudo mais facil.

Sobre design responsivo:

Questão 1

É necessario criar um site dedicado para mobile quando o site de desktop é muito pesado, que pode ocasionar demora no carregamento das funções e uma experiencia ruim para o usuário, ou para sites que utilizam funções exclusivas de aparelhos mobile, como geolocalização e ligações telefonicas.

Não é necessario para um site como o da camara pois o site é basico e leve alem de que não faria benefício de funções exclusivas de aparelhos mobile.

Questão 2

Para preparar a pagina para impressão deve-se definir todas as cores em tons de cinza,sublinhar os links, e de preferência imprimir seus endereços,escolher uma fonte serifada para os textos,definir as unidades de texto em pontos,remover elementos não essenciais, como menus e barras laterais e destacar o endereço do site e o autor do texto.

Sobre multimídia Web:

Questão 1

O elemento source permite o uso dos atributos src(endereço do arquivo de midia), type(o tipo MIME do arquivo) e media(definição do tipo de media pretendido)

O elemento audio permite o uso dos atributos autoplay(o áudio iniciará automaticamente assim que possível sem parar de carregar os dados), autobuffer(o audio será baixado automaticamente, mesmo se não está configurado para reprodução automática),buffered(Um atributo que pode ser lido para determinar os intervalos do áudio que já foram carregados),controls(o navegador oferecerá controles para permitir ao usuário controlar a reprodução do áudio),loop(ao chegar no fim do áudio, ele voltará automaticamente para o começo) muted(o audio iniciará silenciado),played(Um objeto TimeRanges (en-US)indicando que todo o áudio foi reproduzido),preload(dá uma sugestão ao navegador sobre o que o autor pensa que proporcionará uma melhor experiência do usuário)

Questão 2

O elemento source permite o uso dos atributos src(endereço do arquivo de midia), type(o tipo MIME do arquivo) e media(definição do tipo de media pretendido)

O elemento video permite o uso dos atributos autoplay(o video iniciará automaticamente assim que possível sem parar de carregar os dados),buffered(Um atributo que pode ser lido para determinar os intervalos do video que já foram carregados),controls(o navegador oferecerá controles para permitir ao usuário controlar a reprodução do video),height(controla a altura do player), width(controla a largura do player) loop((ao chegar no fim do video, ele voltará automaticamente para o começo),poster(Uma URL indicando uma imagem de prévia do vídeo até o usuário reproduzir ou navegar por ele)

Questão 3

O canvas pode ser usado para criar animações pois é uma api que utiliza o javascript para realiza-las.

As APIs (Application Programming Interfaces) são construções disponíveis nas linguagens de programação que permitem a desenvolvedores criar funcionalidades complexas mais facilmente.API é um conjunto de normas que possibilita a comunicação entre plataformas através de uma série de padrões e protocolos.Por meio de APIs, desenvolvedores podem criar novos softwares e aplicativos capazes de se comunicar com outras plataformas.

Questão 4

O css pode ser usado para alterar o espaço que videos irão tomar, alem de ser usado para fazer media queries, que é a utilização de Media Types com uma ou mais expressões envolvendo características de uma media para definir formatações para diversos dispositivos.

Este video está configurado para ser exibido em 900x560, porem, se o site for exibido em uma tela com ate 480 pixels de largura, ele sera reduzido para 480x300 para ser melhor exibido na tela.
